# simple-interest-cli

A command-line application for calculating simple interest and performing basic arithmetic operations built with Python.

## Project Overview
- **Started:** May 28th, 2026 (after 3 days of Python learning)
- **Last Updated:** May 30th, 2026
- **Status:** Active Development

## Features

### ✅ Completed Improvements (as of May 30th, 2026)

1. **Simple Interest Calculator**
   - Calculate interest based on Principal, Rate, and Time
   - Display both calculated interest and total amount
   - Easy-to-use interactive interface

2. **Menu-Driven Interface**
   - Main menu with options for different operations
   - User-friendly navigation
   - Clean console output

3. **Input Validation & Error Handling**
   - Try-except blocks for robust error handling
   - Validates numeric inputs
   - Provides helpful error messages to users
   - Prevents application crashes from invalid input

4. **Professional Output Formatting**
   - Clear section headers with visual separators
   - Formatted results display
   - Easy-to-read output layout

### 📋 Current Functionality
- Normal calculator option (arithmetic operations)
- Simple interest calculation module
- Exit option with graceful termination

## How to Use

1. Run the application:
   ```bash
   python main.py
   ```

2. Select an option from the menu:
   - Option 1: Basic calculator
   - Option 2: Simple interest calculator
   - Option 3: Exit

3. Follow the prompts to enter required values
